## Quick Start

### 1. Clone the Repository

```bash
git clone https://github.com/WildTrades/NFT_Airdrop_Platform.git
cd NFT_Airdrop_Platform
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Application
```bash
npm run dev
```

### 4. Access the Application

http://localhost:3000
